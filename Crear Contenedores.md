# <ins> **Crear un contenedor por comando**</ins>
* _docker run -dp  &nbsp;&nbsp; --name_ nombrecontenedordedockerhub:version 
   - -d mantiene el contenedor en segundo plano
   - -p conecta los puertos del contenedor con el sistema nativo
 
  # <ins> **En caso de ser un contendor de un sistema al crearlo**</ins>
  * _docker run -it  &nbsp;&nbsp; --name_ nombrecontenedordedockerhub:version bash
    - -i habilita el modo interactivo
    -  -t habilita la terminal y con bash indicas cual
